A firefox extension that helps you to write formulas fast in docs!

All hotkeys for special characters are triggered by: Alt + Shfit + ...
     + M = Mathematics symbols
     + G = Greek letters
     + ArrowUp = superscripts
     + ArrowDown = subscripts
    Special characters: 
      1. Alt + Shift + E = equilibrium symbol
      2. Alt + Shfit + D = degree symbol
