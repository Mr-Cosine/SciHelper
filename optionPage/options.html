Sci‑Helper – Enabled Websites

Your extension will run on the domains listed below:

      Instruction:
    • Paste your url of the website that you want to enable the extension in the box
    • URL must start by https:// or http://
    • It is not necessary to have www, as long as the real url does not
    • * is used to represent text that could be different, following are two examples.
    • If you want to apply to multiple urls that starts with same domain(E.g. both domain/abc and domain/123), use: https://domain/*
    • If you want to apply to multiple urls that shares part of the domain(E.g. abc.domain.com and 123.domain.com), use: https://*.domain.com
    • It is recommended to end the URL with /*
    • If not following the url rule the plugin will not enable as expected
